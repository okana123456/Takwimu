# Takwimu Prototype

Role-based SaaS dashboard prototype for moving organizations away from spreadsheets into a controlled data operations and analytics platform.

## What It Shows

- Role-based platform layout
- Dashboard overview
- Data catalog
- Upload and Google Drive sync concept
- Users and roles matrix
- Task queues for verification and follow-up

## How To Open

Open `index.html` in a browser.

## Suggested Next Steps

- Enable Supabase row-level security before using live client data
- Connect Supabase authentication
- Add organization and role-based permissions
- Connect the cleaned master datasets
- Add Google Drive sync
- Add billing/subscription controls

## Supabase

The frontend reads Supabase connection settings from `supabase-config.js`.
